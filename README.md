## Installation
Github:

    git clone https://github.com/I2rys/HTOB

NPM Packages:

    npm i json-hood && npm i request && npm i chalk
    
## Usage

    node index.js <http/tls> <domain>

 - https/tls - Scan type options.
 - domain - The website domain to scan for https/tls.
